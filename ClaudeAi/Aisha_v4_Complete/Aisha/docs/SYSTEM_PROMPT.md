## 🔧 Customization Tips

### Add More Personal Context
Inject Ajay's memories at the top of the system prompt:
```
[Current memories about Ajay:]
- He is working on a startup in fintech
- His goal for 2025 is to save ₹2 lakhs
- He works best in the mornings
- He has been feeling anxious about his career lately
```

### Force a Specific Mode
Add to the end of the prompt:
```
CURRENT MODE: MOTIVATION — Be extra energetic and powerful in this conversation.
```

### Add Time Context
```
Current time: 9:30 PM IST, Tuesday
[Aisha should be in evening/settling-down energy]
```
