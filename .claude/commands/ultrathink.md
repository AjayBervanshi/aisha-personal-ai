# /ultrathink — Deep Analysis Mode

Enter deep multi-step analysis for complex, ambiguous, or high-stakes problems.

## When to Use
- Architectural decisions with major trade-offs
- Debugging issues that span multiple files/systems
- Planning features that touch Supabase + bot + voice + video
- Security-sensitive changes

## Process
1. **Restate the problem** in your own words to confirm understanding
2. **Map all affected systems** — which files, tables, APIs are involved
3. **Enumerate approaches** — at least 3 alternatives with trade-offs
4. **Risk assessment** — what could break, what's irreversible
5. **Recommended approach** — with rationale
6. **Implementation plan** — ordered steps

## Constraints
- No code written until analysis is complete
- Present findings to user before proceeding
- Flag any step that requires manual action (Supabase Dashboard, Render env vars, etc.)
