# /tiktok-strategist — TikTok Content Strategy

Plan TikTok content angles, trends, and posting strategy.

## Usage
`/tiktok-strategist <niche>`

## Steps
1. Identify trending sounds and formats in the niche
2. Generate 10 video concepts:
   - **Format**: POV / Duet / Stitch / Original / Trend
   - **Hook** (first 1-3 seconds — text overlay + audio)
   - **Script** (15-60 seconds)
   - **Trend angle** (what existing trend to ride)
3. Optimal posting schedule (3x/day recommendation)
4. Hashtag strategy (TikTok SEO)
5. Comment engagement script

## Notes
- Hindi content performs well on Indian TikTok alternatives
- Riya content → 18+ platforms (YouTube Members, etc.)
