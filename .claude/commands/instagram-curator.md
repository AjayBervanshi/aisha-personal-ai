# /instagram-curator — Instagram Content Planner

Plan and draft Instagram content strategy and post calendar.

## Usage
`/instagram-curator <week|month>`

## Steps
1. Generate 7-day (or 30-day) content calendar
2. For each post:
   - **Type**: Reel / Carousel / Static / Story
   - **Hook caption** (first line = stop the scroll)
   - **Caption body** (emotional, relatable)
   - **Hashtag set** (3 buckets: niche × 10, mid × 5, broad × 5)
   - **Best time to post** (IST)
   - **CTA** (follow / save / share / comment)
3. Story sequence for engagement (poll, question, countdown)
4. Highlight cover ideas
